The Way Of Perfect Flask-Vercel Deploy

  • virtualenvでプロジェクトを作成
  • Scriptに移動、.\activate
  • pip install flask
  • pip freeze > requirements.txt
  • cd..
  • vercel dev
  • vercel deploy
  • githubにリモートリポジトリ作成
  • vercel Connect Git Repository
  • github プロジェクトfileアップロード
  • 新しくローカルフォルダを作りvs.codeで開く
  • リモートリポジトリから git clone
  • 変更してコミット
  • プッシュ
